# mulcamp-miniproject
# 프로젝트의 목적
- 본 프로젝트는 서울시 부동산 데이터를 활용하여 대시보드를 개발하는 목적은 부동산 시장 동향을 시각화하고 사용자에게 부동산 투자 결정을 돕기 위함입니다. 이를 통해 사용자는 시장 개요, 가격 및 지역별 동향, 재건축 예측 및 서울시 부동산 시장의 트렌드를 쉽게 파악하고 흐름을 파악할 수 있습니다.
